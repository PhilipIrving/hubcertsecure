Post-Build Checklist
These steps can’t be automated via the API — complete them manually in Monday.com:
1
Remove default group — Each board gets a "Topics" group. Delete or rename it.
2
Set dropdown options — Platform, Structure, Stage, Rank, Category (use the Configure tab above).
3
Connect Boards columns — Link Onboarding and Partnership boards to Clients – Overview.
4
Mirror columns on Clients – Overview — Onboarding Status, Configuration Deadline, Onboarding Progress.
5
Install Power-Ups — Dependency (Onboarding), Progress Tracking (Overview), Files (Partnerships), Button (Overview → GitHub).
6
Build automations — Start with the Go-Live Cascade Workflow.
7
Add validation rules — Status can’t go Active without Go-Live Date, etc.
8
Create saved views — Daily View, Invoicing View, Reports View on Clients – Overview.
9
Resolve invoicing permissions — Confirm visibility on INV: columns before migrating data.
10
Migrate existing clients — Move your 25 active clients into Clients – Overview (or use the Configure tab).
11
Configure GitHub Button — Wire the Button Power-Up to trigger the Report Generator workflow.
